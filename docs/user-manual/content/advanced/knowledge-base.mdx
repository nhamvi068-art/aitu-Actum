# 知识库

知识库是你的个人知识管理空间，支持目录、标签、搜索、导入导出，并可将笔记插入到画布。

## 打开知识库

1. 点击 **工具箱**
2. 找到 **知识库**
3. 点击 **打开** 或 **新窗口**

## 界面结构

- **左侧栏**：搜索、标签筛选、目录树、笔记列表
- **中间**：笔记编辑区（Markdown 编辑）
- **右侧**：相似笔记与知识提取

## 目录与笔记

### 默认目录

- **笔记**：通用知识与灵感记录
- **Skill**：自定义 Skill 的专用目录

### 目录操作

在目录上右键可以：
- 新建笔记
- 复制目录
- 重命名
- 删除

### 笔记操作

在笔记上右键可以：
- 创建副本
- 插入到画布（生成关联卡片）
- 删除

## 标签与检索

- 在顶部搜索框中输入关键词快速定位
- 使用标签筛选笔记
- 在菜单中选择 **管理标签** 进行维护

## 导入与导出

在左上角 **更多操作** 菜单中可以：

- **导入 Markdown**：批量导入 `.md` / `.markdown`
- **导入 ZIP**：导入完整的知识库备份
- **导出 ZIP**：打包导出全部目录、笔记和标签

:::note
导入内容会与现有数据合并，不会覆盖已有笔记。
:::

## 知识提取

在右侧切换到 **知识提取**：

1. 让 AI 归纳关键知识点
2. 选择需要的条目
3. 保存为笔记或合并为一篇

## Skill 目录用法

- **系统 Skill** 会以只读笔记显示
- **自定义 Skill**：在 Skill 目录新建笔记，写清楚角色、输入与输出格式
- 保存后即可在 AI 输入框的 Skill 下拉中选择
