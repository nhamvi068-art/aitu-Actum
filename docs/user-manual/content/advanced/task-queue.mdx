# 任务队列

任务队列用于管理所有 AI 生成任务，支持图片、视频与角色相关任务的追踪、筛选与批量操作。

## 打开任务队列

点击右侧工具栏底部的 **任务队列** 按钮即可展开面板。

## 任务状态

顶部 Tab 支持快速切换：

- **全部** - 查看所有任务
- **生成中** - 正在处理的任务
- **失败** - 需要重试的任务
- **已完成** - 可插入或下载的任务

## 筛选与搜索

- 类型过滤：图片 / 视频 / 角色
- 搜索框：按提示词或内容关键字检索

## 单个任务操作

任务卡片提供常用操作：

- **插入**：将生成结果插入到画布
- **下载**：保存到本地
- **编辑**：进入内置编辑器继续处理
- **重试**：失败任务一键重试
- **删除**：移除任务记录

## 批量操作

点击 **多选** 进入批量模式后，可以：

- **取消** 进行中的任务
- **重试** 失败任务
- **删除** 选中任务

## 常见提示

:::tip
生成完成的图片或视频会自动进入素材库，方便后续复用。
:::
