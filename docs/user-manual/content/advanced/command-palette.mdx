# 命令面板

命令面板可以快速搜索并执行常用功能，适合高频操作和键盘流用户。

## 打开方式

- 快捷键：`Cmd + K`
- 菜单：左上角菜单 → **命令面板**

## 基本用法

1. 输入关键词筛选命令
2. 使用 ↑ / ↓ 选择
3. 按 **Enter** 执行
4. 按 **Esc** 退出

## 命令分类

命令按以下类别归类：

- 工具
- 编辑
- 视图
- AI
- 导出
- 设置

## 使用建议

:::tip
不记得具体位置时，直接在命令面板里搜索功能名称即可。
:::
